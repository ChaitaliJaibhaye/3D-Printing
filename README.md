3D Printing Mini Project – Mould Tray

Project Overview
This repository contains my complete 3D Printing mini project based on the design and fabrication of a Mould Tray.
The mould tray was originally designed using SolidWorks and later exported as an STL file for 3D printing.

Files in This Repository
STL folder
mould_tray.stl – STL file ready for 3D printing

Docs folder
3D Printing.pdf – Complete project report and documentation

README.md – Project description and details

Applications
This 3D-printed mould tray with multiple shapes has applications in:
Chocolate and candy moulding
Creative ice cube trays for cafes and events
Baking such as cookies, cupcakes, and fondant shaping
Soap and butter crafting
Customized product moulding

Tools and Technologies Used
CAD Software: SolidWorks
File Format: STL
Manufacturing Method: Fused Deposition Modeling (FDM)

This project is shared for academic and learning purposes.
